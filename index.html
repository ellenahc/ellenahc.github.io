Overview

In this portfolio, I will go through the learning outcomes I have achieved thus far for the current semester. I am currently studying Software Engineering at Fontys Hogeschool ICT. I am in semester 3. This portfolio, like any other, will be updated periodically.

Learning Outcomes

Each learning outcome (LO) is described below and is accompanied by the work I have done thus far.

LO1: User-Friendly Software

For this LO, I must show how dummy-proof my application is, along with the techniques I used to make it so. Of course, I must also display skills in front and back end development, such as Javascript and ORM.

LO2: Software Quality

I must show that I am capable of using software tools and methods that allows me to continously monitor and improve the software quality of the app during the software development stage.

L03: Software Agility

For this LO, I must choose and implement the most suitable agile software development method for my software project.

LO4: Design and Implementation

I must design and implement a (semi)automated software release process that matches the needs of the project context.

LO5: Culture and Ethics

I must recognize and take into account cultural differences between project stakeholders and ethical aspects in software development.

LO6: Testing Techniques

I must analyze (non-functional) requirements, elaborate architectural designs and validate them using multiple types of test techniques.

LO7: Business Process Analysis

I must analyze and describe simple business processes that are related to my project.

LO8: Professionalism

Speaks for itself (oops!): I must act in a professional manner.
